# 模块划分

## 模块总览

| 模块 | 位置 | 职责 |
|---|---|---|
| 前端 UI | `/userdata/server/www/site/index.html` | 页面展示、交互、调用 API |
| Nginx | 系统服务 | 静态页面托管、API 反向代理、文件加速下载 |
| Flask 后端 | `/userdata/server/apps/filemgr/app.py` | 登录、权限、文件管理、设备控制 |
| 文件存储 | `/userdata/files` | 用户文件真实保存位置 |
| Cloudflared | 系统服务 | 公网隧道和域名访问 |
| eth0 直连配置 | `eth0-direct.service` | 开机配置有线网口固定 IP |
| Windows OpenSSH | 笔记本 | 接收远程关机命令 |
| Wake-on-LAN | 开发板到笔记本 | 发送魔术包唤醒笔记本 |

## 前端 UI 模块

主要能力：

- 登录页面。
- 文件管理页面。
- 概览页面。
- 日志页面。
- 设备控制页面。
- 设置和用户权限页面。

文件管理功能：

- 浏览目录。
- 双击文件夹进入目录。
- 左上角返回上一级。
- 上传文件。
- 上传文件夹。
- 下载文件。
- 下载文件夹。
- Markdown 预览。
- 图片预览。
- PDF 站内预览。
- 删除、重命名、新建文件夹。

## Flask 后端模块

核心职责：

- 只允许访问 `/userdata/files` 内的文件，避免路径穿越。
- 处理登录状态和用户权限。
- 根据权限限制上传、下载、编辑、删除、设备控制。
- 为前端提供 JSON API。
- 调用系统命令或 Python socket 完成设备控制。

典型 API：

| API | 方法 | 作用 |
|---|---|---|
| `/api/login` | POST | 登录 |
| `/api/logout` | POST | 退出登录 |
| `/api/auth/status` | GET | 查询登录状态 |
| `/api/files` | GET | 获取目录文件列表 |
| `/api/upload` | POST | 上传文件或文件夹 |
| `/api/download` | GET | 下载文件 |
| `/api/preview` | GET | 文件预览 |
| `/api/users` | GET/POST | 用户和权限管理 |
| `/api/device/workstation` | GET/POST | 设备配置 |
| `/api/device/workstation/status` | GET | 检测笔记本在线状态 |
| `/api/device/workstation/wake` | POST | 网络唤醒 |
| `/api/device/workstation/shutdown` | POST | 远程关机 |

## Nginx 模块

职责：

- 监听 80 端口。
- 提供 `index.html`。
- 把 `/api/` 反向代理到 Flask。
- 对大文件下载可使用 `X-Accel-Redirect` 加速。

设计原则：

- Nginx 负责传输效率。
- Flask 负责权限和业务逻辑。
- 不把 `/userdata/files` 直接作为普通静态目录暴露。

## Cloudflare Tunnel 模块

职责：

- 把公网域名 `files.jjpersonal.xyz` 映射到开发板本地 `http://localhost:80`。
- 避免家庭宽带没有公网 IP 或路由器端口转发的问题。
- 开机自启，作为系统服务常驻。

## 设备控制模块

分为两个方向：

- 开机：开发板向笔记本有线网卡发送 Wake-on-LAN 魔术包。
- 关机：开发板通过 SSH 登录 Windows，执行 `shutdown /s /f /t 0`。

关键参数：

| 参数 | 当前值 |
|---|---|
| 笔记本有线 IP | `192.168.50.2` |
| 笔记本有线 MAC | `4C:CF:7C:DA:08:56` |
| WOL 广播地址 | `192.168.50.255` |
| SSH 用户 | `woladmin` |
| SSH 端口 | `22` |
| 关机命令 | `shutdown /s /f /t 0` |
